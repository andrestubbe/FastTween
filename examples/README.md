# FastTween Examples

This directory contains standalone examples demonstrating how to use the FastTween module.

## Structure
- **Demo.java**: A comprehensive showcase of easing functions and basic tweening.

## Running the Demo
You can run the demo directly using Maven from this directory:

```bash
# Run the Easing Showcase
mvn compile exec:java -Dexec.mainClass="fasttween.Demo"
```

## Requirements
- **JDK 17+**
- **Maven 3.9+**
- **FastTween**: Ensure the library is built and installed locally (via `mvn install` in the root).
