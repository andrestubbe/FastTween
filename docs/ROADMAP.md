# FastTween Roadmap

## 🎯 Current Status: v0.1.0-alpha

### 🟢 Phase 1: Pure Math (Done)
- [x] Standard Interpolation (Linear, Ease-In, Ease-Out)
- [x] Color space tweening (RGB/HSL)
- [x] Vector2/3 support
- [x] Zero-GC implementation

### 🟡 Phase 2: Native Acceleration (In Progress)
- [ ] JNI SIMD-accelerated bulk tweening (AVX2)
- [ ] FastMath native integration
- [ ] Spline-based path interpolation

### ⚪ Phase 3: Ecosystem
- [ ] Support for complex custom object interpolation
- [ ] Animation state serialization
